Step 1: Download python3 to your computer --------------------------------------------------------------------------------

> To download python3 use below link
> Make sure to download a compatible version for your operating system

> Link: https://www.python.org/downloads/




Step 2: Install python3 ---------------------------------------------------------------------------------------------------

> Install python3 on your computer using the downloaded file


Step 3: Make sure your desktop have successfully install php----------------------------------------------------------------

> Open cmd and type php -v
>If it shows an error, then you have not installed PHP  correctly. and make sure add path to the Environment variable list.



Step 4: Open the terminal for the folder containing the 'server.py' file --------------------------------------------

> Go to the folder contains 'server.py' file
> Click on File Explorer's address bar
> Clear selected address
> Type 'cmd'
> Press Enter (Terminal will popup immediately)




Step 5: Start the server --------------------------------------------------------------------------------------------------

> Type one of the below commands in terminal

>>> python server.py
        OR
>>> py server.py

> Server will start and display,

    Server is listening on http://127.0.0.1:2728




Step 6: Open your browser -------------------------------------------------------------------------------------------------



Step 7: Open the page -----------------------------------------------------------------------------------------------------

> Type http://localhost:2728/ or http://localhost:2728 on your browser
Step 7: Enter marks for all three subjects(not 2 or 1) and press calculate button.
        Then it will show marks for each subject, total marks and average.
